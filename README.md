# Tady.app

_Your local stuff._

Local notes, events, and market on a decentralized network.

[Tady.app](https://tady.app) is a Nostr web client that focuses on events in the physical world. It shows what's going on around your location - automatic or manually selected. You can also define a radius up to 500km.
